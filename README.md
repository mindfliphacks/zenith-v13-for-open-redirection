# ZENITH // V13 Open Redirect Scanner

A Flask-based security tool designed to detect Open Redirect vulnerabilities. It includes a frontend dashboard and several vulnerable lab levels for testing.

## Installation

1. Clone the repository.
2. git clone https://github.com/mindfliphacks/zenith-v13-for-open-redirection.git
3. cd zenith-v13-for-open-redirection
4.  Install dependencies:
   ```bash
   pip install -r requirements.txt
   python3 zenith-v13.py
